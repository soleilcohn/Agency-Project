Agency Portfolio – WRT 307 Service Learning Project  
Last Edit: 2026-04-21  
============================================================

This project was completed by Soleil Cohn as part of a group Agency Project for WRT 307: Writing for the Professions at Syracuse University.

This service-learning project was conducted in collaboration with the Kia Foundation, an organization focused on animal welfare and community engagement. The project aimed to strengthen the foundation’s outreach through clear, engaging, and reusable communication materials.

This repository contains the portfolio website developed to present the team’s work and deliverables.

Live Site:  
https://soleilcohn.github.io/Agency-Portfolio/

----------------------
PROJECT PURPOSE:
----------------------

The objective of this assignment was to design and produce professional communication materials tailored to the needs of a real organization. This included enhancing outreach efforts, increasing audience engagement, and creating content that remains usable beyond the duration of the course.

The project emphasizes:
- Writing for a defined, real-world audience  
- Professional client communication through calls and email  
- Collaborative project development  
- Deliverables with practical, long-term value  

----------------------
WORK COMPLETED:
----------------------

As part of this collaboration, the project team:

- Developed social media graphics and promotional content for outreach  
- Designed informational flyers for the Outdoor Pet Pantry and donation initiatives  
- Created brochures outlining services, mission, and ways to get involved  
- Integrated QR codes for donation access and website navigation  
- Structured materials for clarity, accessibility, and reusability  
- Maintained direct communication with the client to refine messaging  

As an individual contribution, I:

- Designed and developed the portfolio website to present the team’s work  
- Organized project materials into a clear and accessible format  
- Integrated visuals and deliverables into a cohesive digital presentation  
- Ensured usability and navigation aligned with professional standards
